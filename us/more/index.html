What is Andy URL?

Andy URL is an URL redirection system which means if an article url is blog.realandychang.us/username/blog-01, using Andy URL will be shorten into 6 random number and letters.

EX:12ab34.andyurl.top and will redirect to the article URL

Notify the unsecure link inside our webpage

Sometimes creators like to put a link that is outside Andy Chang Blog which could not be monitored, so Andy URL will protect you from harm!

How to define insecure:

-SSL Certificate: No SSL certificate or certificate that will expire within 5 days

-Server: Server have a huge problems such as invasion or other unpredictable problems.

Can I use Andy URL?

Andy URL is currently testing for content creators which means if you became a content creator, you will be able to use Andy URL automatically
